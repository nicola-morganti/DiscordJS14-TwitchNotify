
# Discord Twitch Notify Bot

Discord Twitch Notify Bot, written in discord.js v14.




## Installation

Install the Discord Bot on your Computer


### Windows
```bash
  ren .env.example .env
  npm install
```

Go to the .env and edit it with your bot data.

```bash
  node .
```
    
### MacOS / Ubuntu
```bash
  mv .env.example .env
  npm install
```

Go to the .env and edit it with your bot data.

```bash
  node .
```
